# Vivek Shukla Portfolio

A modern, responsive personal portfolio website showcasing projects, skills, and experience with an elegant dark theme design.

## 🎨 Features

- **Modern Design**: Dark theme with vibrant gradient accents and smooth animations
- **Responsive Layout**: Fully responsive design that works on desktop, tablet, and mobile devices
- **Smooth Animations**: CSS animations including typing effects, zoom effects, and floating geometric shapes
- **Project Showcase**: Display of projects with tech stack information
- **Skills Section**: Interactive progress bars showing skill proficiency
- **Tech Stack Icons**: Visually appealing icons for technologies used
- **Optimized Performance**: Minified assets and lazy-loading capabilities
